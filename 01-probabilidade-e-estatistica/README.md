Esta etapa do bootcamp teve como foco a base estatística usada no dia a dia de quem trabalha com dados: estatística descritiva, probabilidade, amostragem e testes de hipótese. Cada pasta abaixo corresponde a um desafio prático aplicado sobre um dataset diferente, com Python (Pandas, NumPy, Matplotlib, Seaborn, SciPy) em notebooks no Google Colab.

## 📂 Projetos

| # | Projeto | Dataset | Tema |
|---|---------|---------|------|
| 1 | [Python para Dados](./1.Python-para-Dados/) | Saúde do sono e estilo de vida | Manipulação de dados com Pandas + medidas de tendência central |
| 2 | [Frequências e Medidas](./2.frequencias-e-medidas/) | Notas do ENEM 2023 (fictício) | Estatística descritiva, quartis, histogramas e outliers |
| 3 | [Probabilidade e Amostragem](./3.Probabilidade-e-Amostragem/) | Dados populacionais brasileiros | Probabilidade, distribuição normal/binomial e amostragem |
| 4 | [Teste de Hipótese](./4.Teste-de-Hipotese/) | Teste A/B + dados de pacientes cardíacos | Dimensionamento amostral, teste t, ANOVA, qui-quadrado e Shapiro-Wilk |
